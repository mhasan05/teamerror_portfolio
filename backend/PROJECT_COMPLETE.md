## ğŸ³ Docker Containerization

The project has been fully dockerized with separate configurations for development and production environments:

### Development Setup
1. Clone the repository
2. Create environment files from examples
3. Run `docker-compose up --build`
4. Access:
   - Frontend: http://localhost:3000
   - Backend API: http://localhost:8000/api/
   - Admin Panel: http://localhost:8000/admin/

### Production Deployment
1. Configure production environment files
2. Run `docker-compose -f docker-compose.prod.yml up --build`
3. Set up DNS and SSL certificates
4. Configure reverse proxy if needed
